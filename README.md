# Calendario Pensione Cani

Un'applicazione React TypeScript per calcolare le tariffe della pensione cani con calendario interattivo.

## Funzionalità

- 📅 Calendario interattivo per selezionare date di inizio e fine
- 💰 Calcolo automatico tariffe stagionali
- 🐕 Supporto per più cani (sconto 50% dal secondo in poi)
- 🔥 Opzione box riscaldato (+5€/giorno)
- 🏊 Opzione piscina (+25€ totale)
- 📱 Interfaccia mobile-friendly in italiano
- 🔄 Pulsante reset per cancellare tutti i campi

## Tariffe

- 15/01 - 30/07: €20/giorno
- 01/08 - 15/09: €25/giorno  
- 16/09 - 19/12: €20/giorno
- 20/12 - 14/01: €25/giorno
- 24/12 - 25/12: €100/giorno
- 31/12 - 01/01: €100/giorno
